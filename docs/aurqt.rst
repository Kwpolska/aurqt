SYNOPSIS
========

*aurqt*

DESCRIPTION
===========

aurqt is an AUR *manager*.  Unlike its older brother, PKGBUILDer, it is a
graphical (Qt) tool, used to work with AUR, used in place of the AUR web
interface (aurweb).

COMMAND-LINE ARGUMENTS
======================
